# TabBar 图标说明

请在此目录下添加以下8个PNG图标文件（建议尺寸：81x81像素）：

## 需要的图标文件：

### 课程
- `course.png` - 课程图标（未选中状态，灰色）
- `course-active.png` - 课程图标（选中状态，蓝色）

### 消息
- `message.png` - 消息图标（未选中状态，灰色）
- `message-active.png` - 消息图标（选中状态，蓝色）

### 社区
- `forum.png` - 社区图标（未选中状态，灰色）
- `forum-active.png` - 社区图标（选中状态，蓝色）

### 我的
- `mine.png` - 我的图标（未选中状态，灰色）
- `mine-active.png` - 我的图标（选中状态，蓝色）

## 图标建议：
- 课程：📚 书本/课本图标
- 消息：💬 消息/聊天气泡图标
- 社区：👥 人群/社区图标
- 我的：👤 人物/用户图标

## 获取图标资源：
可以从以下网站获取免费图标：
- https://www.iconfont.cn/
- https://icons8.com/
- https://www.flaticon.com/

注意：微信小程序tabBar图标大小限制为40KB以内
