# 系统监控模块 - 快速启动指南

## 🚀 5分钟快速上手

### 第一步：执行数据库脚本（1分钟）

1. 打开MySQL客户端或命令行
2. 选择你的数据库
```sql
USE your_database_name;
```

3. 执行脚本
```sql
source E:\codes\idea workspace\Mobile-Classroom\sql\proj_fz_system_monitor.sql;
```

或者直接复制SQL内容到MySQL客户端执行。

**验证**：执行成功后会看到创建了3个表和插入了8条示例数据。

---

### 第二步：启动后端服务（2分钟）

```bash
# 方式1：使用命令行
cd E:\codes\idea workspace\Mobile-Classroom\ruoyi-admin
mvn spring-boot:run

# 方式2：使用IDEA
# 右键 RuoYiApplication.java -> Run 'RuoYiApplication'
```

**验证**：看到 "Started RuoYiApplication" 表示启动成功。

---

### 第三步：启动前端服务（1分钟）

```bash
cd E:\codes\idea workspace\Mobile-Classroom\ruoyi-ui
npm run dev
```

**验证**：看到 "Compiled successfully" 表示启动成功。

---

### 第四步：访问系统（1分钟）

1. 打开浏览器访问：http://localhost:80
2. 登录系统（使用若依默认账号）
3. 在左侧菜单找到 **"系统监控"** 模块
4. 点击进入各个子菜单：
   - 📊 监控统计（推荐首页）
   - 💻 服务器监控
   - 🗄️ 数据库监控
   - 📝 监控记录

---

## 🎯 快速功能演示

### 查看服务器实时状态
1. 点击 **系统监控 > 服务器监控**
2. 查看CPU、内存、磁盘等实时指标
3. 观察下方的趋势图表
4. 点击 **刷新数据** 按钮获取最新数据

### 查看数据库性能
1. 点击 **系统监控 > 数据库监控**
2. 查看连接数、响应时间等指标
3. 查看慢查询统计图表

### 查看监控记录
1. 点击 **系统监控 > 监控记录**
2. 使用筛选条件查询数据
3. 点击 **详情** 查看完整信息
4. 对告警记录进行 **处理**
5. 点击 **导出** 下载Excel报表

### 查看统计分析
1. 点击 **系统监控 > 监控统计**
2. 查看统计卡片和图表
3. 使用快捷操作按钮快速跳转
4. 点击 **立即采集** 手动触发监控

## ⚠️ 注意事项

1. **首次访问需要权限配置**
   - 如果菜单不显示，请在若依系统管理中配置权限
   - 权限标识：proj_fz:monitor:*

2. **监控数据会自动积累**
   - 建议定期清理30天前的历史数据
   - 或配置定时任务自动归档

3. **图表需要ECharts支持**
   - 已自动安装ECharts 5.4.0
   - 如图表不显示，请检查浏览器控制台

4. **服务器监控需要时间**
   - 首次加载CPU数据需要等待1秒（OSHI采集周期）

## 🆘 遇到问题？

### 页面打不开
- 检查前端服务是否启动
- 检查路由配置是否正确
- 清除浏览器缓存重试

### 数据加载失败
- 检查后端服务是否启动
- 检查数据库表是否创建
- 打开浏览器F12查看网络请求

### 图表不显示
- 检查浏览器控制台是否有错误
- 确认ECharts是否正确安装
- 刷新页面重试

### 权限不足
- 联系管理员配置权限
- 确认登录用户有相应角色
